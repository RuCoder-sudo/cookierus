Простой способ убедиться, что ваш сайт соответствует законодательству России о файлах cookie. Красивый баннер, логирование согласий, генератор политики.

== Description ==

CookieRus — плагин для получения и управления согласием пользователей на использование файлов cookie на вашем сайте WordPress. Версия 1.0.5.

**Возможности плагина:**

* Настраиваемый баннер согласия с файлами cookie с кнопками «Принять», «Отклонить», «Настроить»
* Иконка cookie в баннере с настройкой размера
* Дополнительная произвольная кнопка с кастомным URL
* Расширенные настройки стилизации: шрифт, тени, радиус кнопок, цвет ссылок, эффекты наведения
* Режим сворачивания баннера в плавающую иконку cookie
* Настройка повторного показа (никогда / 7 дней / 30 дней / каждый раз)
* Категории согласия: Необходимые, Функциональные, Аналитика, Производительность, Реклама
* Менеджер логов согласий с экспортом в CSV
* Генератор политики в отношении файлов cookie
* Автообновление через GitHub Releases
* Совместимость с WooCommerce

**Новое в версии 1.0.5:**

* Иконка cookie в баннере — тумблер включения + выбор размера.
* Дополнительная кнопка с произвольным URL в баннере.
* Расширенные настройки: шрифт, тень баннера, закругление кнопок, эффекты наведения, цвет ссылки политики.
* Режим сворачивания баннера в круглую плавающую иконку cookie.
* Настройка повторного показа баннера.
* Автообновление плагина напрямую из GitHub.
* Бейдж версии в шапке страницы настроек.

**Совместимость:** WordPress 5.0 – 6.7 · PHP 7.4 – 8.3 · WooCommerce

== Installation ==

1. Загрузите папку `cookierus` в директорию `/wp-content/plugins/`
2. Активируйте плагин через меню «Плагины» в WordPress
3. Перейдите в раздел **CookieRus** в меню администратора
4. Настройте внешний вид и контент баннера на вкладке «Настройки баннера»
5. Сохраните изменения и проверьте баннер на сайте

== Frequently Asked Questions ==

= Будет ли баннер мигать при переходах между страницами? =

Нет. Плагин проверяет наличие cookie согласия на стороне PHP ещё до рендеринга баннера, поэтому мерцание полностью отсутствует.

= Как работает режим сворачивания? =

Если включён тумблер «Разрешить сворачивать баннер», пользователь может нажать на кнопку закрытия (×), и баннер свернётся в небольшую круглую иконку cookie в нижнем углу экрана. При клике на иконку баннер возвращается.

= Где хранятся логи согласий? =

Все данные хранятся в базе данных WordPress в таблице `wp_cookierus_logs`. Их можно экспортировать в CSV или очистить через вкладку «Менеджер логов».

= Как настроить дополнительную кнопку? =

В «Настройках баннера» есть раздел «Дополнительная кнопка». Включите тумблер, введите текст кнопки, URL и настройте цвета.

= Как обновить плагин? =

Плагин автоматически проверяет обновления из репозитория GitHub и показывает уведомление в стандартном меню WordPress «Плагины», как и любой плагин из каталога WordPress.
