# Changelog для CookieRus

Все значимые изменения в этом проекте будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.5] - 2026-05-29

### Added
- **Иконка cookie в баннере** — настройка отображения иконки cookie перед текстом баннера с выбором размера (маленький / средний / большой).
- **Дополнительная кнопка в баннере** — настройка произвольной кнопки с кастомным текстом, URL, цветом фона и текста.
- **Расширенные настройки стилизации** — закругление кнопок (0–20 px), размер шрифта кнопок (12–18 px), цвет ссылки политики, выбор шрифта (inherit / sans-serif / serif / mono), тень баннера (нет / лёгкая / средняя / тёмная), эффект при наведении (нет / осветление / затемнение / масштаб).
- **Режим сворачивания баннера** — тумблер «Разрешить сворачивать баннер». При нажатии кнопки закрытия баннер сворачивается в круглую плавающую иконку cookie в углу экрана. Повторный клик — баннер возвращается.
- **Настройка повторного показа** — опция определяющая, когда баннер показывать снова: никогда / через 7 дней / через 30 дней / при каждом посещении.
- **Автообновление через GitHub** — плагин проверяет обновления из репозитория https://github.com/RuCoder-sudo/cookierus напрямую через WordPress.
- **Значок версии в шапке админки** — бейдж `v1.0.5` отображается рядом с заголовком страницы настроек.

### Changed
- Версия плагина обновлена до 1.0.5 во всех файлах.
- Plugin URI обновлён до https://github.com/RuCoder-sudo/cookierus.
- Tested up to: WordPress 6.7.

### Fixed
- Мерцание баннера при переходах между страницами устранено: PHP-проверка куки до рендера + CSS `display:none` по умолчанию.
- Кнопка «Очистить все логи» теперь корректно защищена nonce.

## [1.0.4] - 2026-05-20

### Added
- Баннер не мигает при переходах между страницами (PHP-проверка cookie до рендера).
- Кнопка «Очистить все логи» в менеджере логов с защитой nonce.
- README.md переведён на русский язык с emoji-форматированием.

## [1.0.3] - 2025-06-01

### Added
- Предпросмотр баннера в режиме реального времени в настройках.
- Настройка анимации появления: Fade In, Slide, Bounce.
- Поддержка позиции баннера: снизу, сверху, снизу-слева, снизу-справа.

## [1.0.2] - 2025-03-15

### Added
- Логирование действий пользователей с файлами cookie (принял/отклонил).
- Экспорт логов в CSV-формат.
- Генератор политики в отношении файлов cookie.

## [1.0.1] - 2024-12-01

### Added
- Настройка категорий согласия: Необходимые, Функциональные, Аналитика, Производительность, Реклама.
- Модальное окно настройки согласия по категориям.

## [1.0.0] - 2024-09-01

### Added
- Первый релиз плагина CookieRus.
- Баннер согласия с файлами cookie с кнопками «Принять» и «Отклонить».
- Настраиваемый дизайн баннера (цвета, шрифты, позиция).
- Совместимость с WordPress 5.0+ и PHP 7.4+.
